# Marketing Data Analysis using Python and Machine Learning

## Project Overview
This project analyzes marketing campaign performance using Python and Machine Learning. It helps identify the best-performing marketing channels by analyzing advertising spend, impressions, clicks, conversions, and revenue.

## Objective
- Analyze marketing campaign performance.
- Visualize important business insights.
- Predict revenue using a Linear Regression model.
- Evaluate the machine learning model.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset
Marketing Data Analysis Dataset provided by YBI Foundation.

## Machine Learning Model
- Linear Regression

## Project Workflow
- Data Collection
- Data Exploration
- Data Cleaning
- Data Visualization
- Feature Selection
- Train-Test Split
- Model Training
- Prediction
- Model Evaluation

## Conclusion
This project demonstrates how data analysis and machine learning can be used to evaluate marketing campaigns and predict future revenue, helping businesses make better marketing decisions.
